# Movie Recommendation System

A content-based movie recommendation engine built using **Python** and **Scikit-Learn**.

---

##  Project Status
> **Current Focus:** I am actively working on implementing and deploying this recommendation engine as a fully functional application. 

---

##  Datasets & Weights

Due to GitHub's file size limits, the precomputed similarity matrices are hosted externally. If you are cloning this repository to run it locally, please download the following files and place them directly in your project's root directory:

* [Download movies_dict.pkl](https://drive.google.com/file/d/1OZLjY_UvdiB2cyVx7DWqNNLzbzoMC7FX/view?usp=sharing)
* [Download similarity.pkl](https://drive.google.com/file/d/1sypCwcn3mLXy39QpXVR8I3FOInLyzuDc/view?usp=sharing)
