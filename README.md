## Note on Datasets & Weights
Due to GitHub's file size limits, the precomputed similarity matrices are hosted externally. If running locally, please download them and place them in the root directory:

* [Download movies_dict.pkl](https://drive.google.com/file/d/1OZLjY_UvdiB2cyVx7DWqNNLzbzoMC7FX/view?usp=sharing)
* [Download similarity.pkl](https://drive.google.com/file/d/1sypCwcn3mLXy39QpXVR8I3FOInLyzuDc/view?usp=sharing)

The app is configured to automatically pull these files when deploying to Streamlit Cloud.
